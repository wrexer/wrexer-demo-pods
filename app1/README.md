# App1: Simple Status Service

A lightweight Express.js application designed as a demonstration service.

## Features
- **Static Hosting**: Serves the frontend from the `public/` directory.
- **Status API**: Provides a simple health check endpoint.

## API Endpoints
- `GET /api/status`: Returns current server status, timestamp, and version.

## Tech Stack
- **Backend**: Node.js, Express.js
- **Frontend**: HTML/CSS (Static)
