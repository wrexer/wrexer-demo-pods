# 🚀 Wrexer Service Demos

Welcome to the **Wrexer Service Demos** repository! This collection contains various demo applications designed to help you understand how to build and deploy services on the [Wrexer Platform](https://www.wrexer.com).

## 🌍 About Wrexer

Wrexer is a powerful platform for hosting and managing your web applications and services. These demos showcase the simplicity and flexibility of launching apps on our infrastructure.

## 📺 Resources

- **Demo Videos**: Find step-by-step guides and walkthroughs on our [YouTube Channel](https://www.youtube.com/@wrexer).
- **Official Website**: [www.wrexer.com](https://www.wrexer.com)

## 🛠️ Getting Started

You can use these apps as a starting point for your own projects:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/wrexer/wrexr-demo.git
    ```
2.  **Explore the Apps**: Check out `app1` and `app2` for different service implementations.
3.  **Customize**: Edit the code according to your specific needs.
4.  **Build Your Image**:
    ```bash
    docker build -t your-username/your-app-name .
    ```

## 🚢 Deploying to Wrexer

Once your application is ready, you can host it securely on Wrexer:

- **Public Repositories**: Easily launch services directly from public Docker images.
- **Private Repositories**: To host private images, ensure you add your **Docker Hub Username** and **Token** in the [Wrexer Settings](https://www.wrexer.com/settings).
